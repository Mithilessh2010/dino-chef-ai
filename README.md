# Dino Chef

Dino Chef is a dinosaur-themed website that helps users find recipes based on the ingredients they already have. The goal of the site is to make cooking easier, faster, and more fun without needing to buy extra food.

---

## What Is Dino Chef?

Dino Chef allows users to enter ingredients from their kitchen and receive recipe suggestions that use those ingredients. Instead of searching online for recipes and missing items, Dino Chef shows meals that are possible right now.

---

## Features

- Search for recipes using available ingredients  
- Simple and easy-to-use interface  
- Dinosaur-themed design  
- Works on desktop and mobile devices  

---

## How It Works

1. The user enters the ingredients they have.  
2. The website processes the ingredients.  
3. Matching recipes are displayed.  
4. The user chooses a recipe and starts cooking.

---

## Purpose

Dino Chef was created to help reduce food waste and make cooking less stressful. It encourages users to use what they already have instead of buying new ingredients.

---

## Future Improvements

- Add dietary filters (vegetarian, vegan, etc.)  
- Allow users to save favorite recipes
